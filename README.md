## 💡 The Problem

> A delivery business owner in the UK spends $800/month on fuel — but has no idea where it goes.
> A daily commuter fills up every 3 days — but never tracks the actual monthly cost.
> A fleet manager guesses fuel budgets — and always gets it wrong.

**Most people have no visibility into one of their biggest regular expenses — fuel.**

Fuco fixes that. A complete Flutter fuel intelligence app — live prices, trip cost calculator, Google Maps navigation, and full expense history — all in one place.

## 📱 Core Features

### ⛽ Live Fuel Prices
- Real-time **Petrol, Diesel, and CNG** price updates
- Always know the current price before you fill up
- Price history to track fuel cost trends over time

### 🚗 Vehicle Management
- Add multiple vehicles — car, bike, truck, any type
- Set fuel type and average mileage per vehicle
- All trip calculations auto-use your vehicle's specs

### 🧮 Trip Fuel Cost Calculator
- Enter **origin + destination** or distance in km
- App calculates **exact fuel cost** for the trip
- Uses your vehicle's mileage + live fuel price = pinpoint accuracy
- Know the cost before you start the engine

### 🗺️ Google Maps Navigation
- Built-in Google Maps for **turn-by-turn navigation**
- Automatically calculates route distance
- Feeds distance directly into fuel cost calculator

### 📊 Expense Tracker & History
- **Daily view** — today's fuel spend
- **Weekly view** — this week at a glance
- **Monthly view** — full month breakdown
- **Trip history** — every logged journey with cost, distance, fuel used
- Visual charts showing spending trends over time
